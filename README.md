# 🚀 DSA Problem Tracker

An intuitive, interactive web application designed to help computer science students organize, log, and track their Data Structures and Algorithms (DSA) practice problems efficiently. 

---

## ✨ Features

* **Problem Logging:** Add and categorize LeetCode or custom DSA problems by topic (e.g., Arrays, Graphs, Dynamic Programming).
* **Status Tracking:** Keep tabs on problem status (e.g., Solved, In Progress, Needs Review).
* **Performance Insights:** A structured dashboard view to monitor practice consistency and mastery over time.
* **Responsive UI:** Clean, dark-mode friendly user interface built for seamless navigation.

## 🛠️ Tech Stack

* **Frontend Framework:** React (with Vite)
* **Language:** TypeScript (ensuring type safety and robust code)
* **Styling:** Tailwind CSS (for modern, utility-first responsive styling)
* **Linting & Code Quality:** ESLint
